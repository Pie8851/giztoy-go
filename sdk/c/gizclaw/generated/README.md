# Generated GizClaw C RPC Bindings

This directory is generated by `tools/gzc-rpcgen`. Do not edit generated files by hand.

Regenerate from the repository root with:

```sh
go run ./tools/gzc-rpcgen -schema api/rpc.json -include api -out sdk/c/gizclaw/generated -package gzc -format
```

Generated files depend only on the C SDK runtime headers under `sdk/c/gizclaw/include`.
