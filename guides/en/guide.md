# Project Guidelines

GizClaw is an out-of-the-box Agent Runtime and Edge Server for GizClaw devices. It uses WebRTC as the connection and transport layer between devices, nodes and clients, and provides services such as Agent Runtime, Admin/RPC API, device telemetry and status monitoring, OTA, digital content, social and Gameplay on it.
