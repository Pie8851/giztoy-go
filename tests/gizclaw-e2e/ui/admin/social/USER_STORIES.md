# Admin UI Social User Stories

- As an admin operator, I can browse top-level Friends and open a friend detail page without going through Peer Detail.
- As an admin operator, I can browse top-level Contacts and open a contact detail page.
- As an admin operator, I can create a direct friend relation and inspect the backing workspace history panel.
- As an admin operator, I can browse Friend Groups and open a group detail page.
- As an admin operator, I can create a Friend Group, manage members, manage invite tokens, and inspect the History tab.
