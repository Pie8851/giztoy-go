# Play UI Social

- As a Play UI user, I can manage my peer-local Contacts address book.
- As a Play UI user, I can open Friends and manage invite-token based direct friend flows without the old request/approval surface.
- As a Play UI user, I can open Groups and join by invite token without a group request list.
- As a Play UI user, I can open the Chat drawer and see that social chat is backed by workspace history and voice controls.
