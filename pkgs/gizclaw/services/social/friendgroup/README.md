# Friend Group Domain

Tracking issue: https://github.com/GizClaw/gizclaw-go/issues/23

This package is reserved for friend groups, friend group membership, and friend
group messages.

Planned scope:

- `friend_group.{list,get,create,put,delete}`
- `friend_group.members.{list,add,put,delete}`
- `friend_group.messages.{list,get,send}`
